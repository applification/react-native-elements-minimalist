---
title: Installation
---

```sh
npm install --save react-native-minimalist
```

## Usage

```javascript
 import {
  Content,
  HeaderText,
  BodyText
} from 'react-native-minimalist';

<Content>
  <HeaderText>Big header with padding</HeaderText>
  <BodyText>Body text with padding</BodyText>
</Content>
```
